Google Maps API Key Gerekli

Haritayı görmek için aşağıdaki alana Google Maps API anahtarınızı girin. API key'i application.properties dosyasına da google.maps.api-key=BURAYA_KEY şeklinde ekleyebilirsiniz.

Geliştirme ortamı için: Google Cloud Console → Maps JavaScript API → Enable → Credentials → API Key
